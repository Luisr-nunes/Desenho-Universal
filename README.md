## 📋 Sobre este Repositório

Este repositório reúne todos os arquivos necessários para a realização da **Atividade Prática (AV1)** da disciplina de **Desenho Universal**, do curso de **Arquitetura e Urbanismo** do Grupo SER Educacional.

Aqui você encontrará os desenhos técnicos do projeto de referência utilizados na atividade, além do guia passo a passo com as instruções completas.

## 🎯 Objetivo da Disciplina

A disciplina de **Desenho Universal** aborda os princípios e normas para o desenvolvimento de projetos arquitetônicos acessíveis e inclusivos, seguindo as diretrizes da **NBR 9050** e os **7 princípios do Desenho Universal**, garantindo que espaços sejam funcionais para todas as pessoas, independente de suas capacidades.

---

## 📌 Como Utilizar

1. Leia atentamente o arquivo **"PASSOS PARA A AV1"** antes de iniciar
2. Faça o download dos arquivos necessários para a sua atividade
3. O arquivo `.dwg` pode ser aberto no **AutoCAD** ou em qualquer software compatível
4. Consulte as pranchas em PDF como referência visual do projeto
